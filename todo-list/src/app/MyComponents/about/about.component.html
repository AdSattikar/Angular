About Todo-List

This is a simple todo List created using Angular

It utilizes the core Angular Concepts such as:

  1. Components: todos,todo-item,add-todo,about
  2. Services: addTodo, ngFor and ngIf directives
  3. Dependency injection: the todos are dependent on todo-item and hence the same are injected into todos array using the Component Injection method
  4. Event Emitters and Event Binding: Each todo calls either addTodo service and binds the data and its description to the todos array
  5. Routing and navigation: Respective routes are created for home and about page and navigated through app-routing.module.ts
